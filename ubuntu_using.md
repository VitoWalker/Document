6.复制文件到/等目录下

sudo cp / usr / share / zoneinfo /亚洲/上海/ etc / localtime

7.更新时间

sudo ntpdate time.windows.com





修改时间

```
sudo date -s MM / DD / YY // 修改日期 
sudo date -s hh：mm：ss // 修改时间
```

在修改时间以后，修改硬件CMOS的时间

```
sudo hwclock --systohc // 非常重要，如果没有这一步的话，后面时间还是不准
```
